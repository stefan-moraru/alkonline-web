Documentatia limbajului Alk

Este un limbaj de programare C-like
Comentarii
/*

comentariile se scriu asa

*/
Programele se ruleaza astfel
krun programul.alk -cINIT=".Map" sau cu configuratii intiale, de exemplu vectorul a este [3, 5, 6, 4]:

krun programul.alk -cINIT="a |-> [3, 5, 6, 4]"
Structura unui program
functia1(argumente1) {

return valoare1;

}

functia2(argumente2) {

return valoare2;

}

si programul principal ce contine instructiuni si apelarea unor functii

instructiuni

apelare functii
Instructiunea if
Arata ca in C: if (conditie) instructiune1 sau

if (conditie) instructiune1; else instructiune2
Instructiunea for
Se foloseste operatorul ++ astfel: ++i (nu i++)

for (i= 1; i <= n; ++i) instructiune
Instruciunea while
Instructiunea se pune intre { si }

while (conditie)
{
Instructiune;
}

Instructiunea do { } while nu exista
Tipuri de date
Tipurile de date nu se precizeaza, nici in apeluri, nici ca parametri formali, se subinteleg din context.
Tablouri
a = [3, 5, 6, 4]; declara un vector cu 4 numere intregi

al i-lea element din vector a[i]
